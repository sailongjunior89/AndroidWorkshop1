# Android Workshop Portfolio

## Overview

This repository contains a collection of Android application projects developed to explore and demonstrate core Android development concepts using Kotlin and Android Studio.

The projects cover Android fundamentals, UI design, threading, services, broadcasts, media handling, camera integration, and application architecture. Together, they showcase hands-on experience with common Android components and lifecycle management.

## Technologies

* Kotlin
* Android SDK
* Android Studio
* XML Layouts
* RecyclerView
* Fragments
* Services
* Broadcast Receivers
* Threads & Concurrency
* Camera & Media APIs
* ConstraintLayout
